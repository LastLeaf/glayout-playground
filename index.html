HTML （支持 div span img）
CSS （支持的选择器和样式见底部的说明）

这不是一个普通的 HTML Playground 。使用 F12 ，你会发现 HTML 内容是画在一块 Canvas 上的！

这里使用了 glayout 进行 2D 界面排版绘制，它是一个 rust / WebAssembly 库，在浏览器中基于 WebGL 绘制。去 GitHub ！

目前支持的 HTML 标签：

  • div
  • span
  • img src="URL"

目前支持的 CSS 选择器：

  • class 选择器
  • ID 选择器
  • 标签名选择器

目前支持的 CSS 样式：

  • display（可用值 block inline，和有限的 flex inline-block）
  • box_sizing width height（长度单位支持 px in cm mm em vw vh 百分比和微信小程序风格的 rpx）
  • position（可用值 static relative absolute）
  • top right bottom left
  • 各个方向的 margin border padding
  • flex（有限的 flex-grow flex-shrink）
  • font font-family font-size line-height text-align
  • opacity color background-color
